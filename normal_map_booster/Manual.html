Normal Map Booster

A Krita plugin that allows you to generate custom normal maps from selected layers.

Usage

Go to Settings -> Configure Krita -> Python Plugin Manager and enable "Normal Map (Edge Detection)"

Select a layer and set normal map properties.

Click the "New Normal Map" button to generate a new normal map.

Click the "Export" button to export a texture.

Normal map properties

Filter Type - determines the type of the edge detection that is used to create a new normal map

Normal Map Format - [DirectX/OpenGL]

Width - the width of the edges (the radius of the edge detection)

Shape - [bump(convex)/hole(concave)], assuming black pattern on a white background

Slope - determines the direction of the surface normals; enhances texture details
